Clone the project

```bash
  git clone https://github.com/whyshashi/pp-assignment.git
```

Go to the project directory

for assignment1

```bash
  cd assignment1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
